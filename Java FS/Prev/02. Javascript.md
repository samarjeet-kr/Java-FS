### Functions
A function is a block of code that performs a specific task.
Its like a recipe you define it once and you can use it multiple times.

Types :
1. Named function
2. Anonymous function
3. Arrow function

Key characteristic :
1. Reusable : write once, use multiple times.
2. Modular : break down large code into smaller, into manageable pieces

```js
function greet(name) {
	console.log(`Hello, ${name}!`);
}
greet('John'); // Output: Hello, John!
```

### DOM (Document Object Model)
- DOM is a programming interface for HTML and XML.
- It represents the structure of a webpage as a tree like data structure.
- Allowing scripts to interact with and manipulate the pages content.

Key Concepts : 
1. *Nodes* : elements, attributes and texts are all nodes in the DOM tree.
2. *Elements* : HTML tags like `<div>`,`<p>` etc.
3. *Attributes* : Key-Value pairs like `ID = "header"`

```js
document.getElementById('header')
// id or class
document.querySelector('h1');
document.querySelectorAll('p');
```

### Why DOM is Important ?
1. Dynamic content
2. Updated webpages without reloading
3. Interactive WebApps
4. Respond to user event, like clicks and keyboard inputs
5. Responsive design : adapts layouts to different screen sizes.
